# Приклади моїх проектів (Portfolio)

У цьому репозиторії я зібрав частини проектів та мікросервісів, які розробляв для реальних завдань. Це допоможе оцінити мій підхід до архітектури, написання коду та вирішення технічних проблем.

Тут представлені напрацювання в автоматизації, роботі з AI-сервісами, парсингу даних та побудові бекенд-логіки.

## Що тут є:

*   **articles-stati** — Автоматизований контент-пайплайн. Генерує статті через Anthropic API, перевіряє їх на унікальність, створює зображення та автоматично публікує в WordPress.
*   **parser-indeed** — Скрапер для збору вакансій з Indeed. Працює з проксі, обробляє великі обсяги даних та зберігає результати для аналітики.
*   **monitoring-of-competitors** — Скрипти для моніторингу активності конкурентів: відстеження цін та оновлень на сайтах.
*   **python-transcription-service** — Мікросервіс для перетворення аудіо в текст (STT) за допомогою AI моделей.
*   **voice-assistants** — Розробка та інтеграція голосових асистентів для платформи Оки-Токи (VoIP, IVR логіка).
*   **tts-service** — Порівняння та реалізація синтезу мовлення (Text-to-Speech) через Azure та OpenAI.
*   **python-microservice-repo** — Приклад структури репозиторію для проектів на базі мікросервісів.

**Важливо:** Усі API-ключі, паролі та конфіденційні дані були видалені. Деякі скрипти можуть не запускатися без відповідних змінних оточення.
