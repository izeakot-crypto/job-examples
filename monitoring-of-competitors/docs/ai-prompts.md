# AI Промпти для моніторингу конкурентів

## 1. Суммаризація новин з блогу

```
Ти - аналітик конкурентів в сфері VoIP та contact center рішень.

Проаналізуй нові статті з блогу компанії {{ company }}:

{% for post in blogPosts %}
Стаття {{ loop.index }}:
Заголовок: {{ post.title }}
Дата: {{ post.publishedAt }}
Опис: {{ post.excerpt }}
{% endfor %}

Створи короткий саммарі (до 300 слів):

1. **Основні теми:** Які напрямки покриває компанія в своїх публікаціях
2. **Нові функції:** Чи згадуються нові продукти або фічі
3. **Маркетингова стратегія:** На що робиться акцент (ціна, якість, інновації)
4. **Цільова аудиторія:** Для кого призначений контент (малий бізнес, enterprise, конкретні індустрії)
5. **Insights:** Що може бути корисним для нашої компанії

Формат відповіді: structured markdown з чіткими секціями.
```

## Налаштування в n8n

### Code node для формування промпту:

```javascript
// Приклад формування промпту для аналізу компанії
const company = $json.company;
const websiteData = $json.websiteData;
const blogPosts = $json.blogPosts || [];
const socialData = $json.socialData || {};

// Форматуємо дані для промпту
const blogPostsList = blogPosts.map((post, index) =>
  `Стаття ${index + 1}:\nЗаголовок: ${post.title}\nДата: ${post.publishedAt}\nОпис: ${post.excerpt}`
).join('\n\n');

const prompt = `
Ти - аналітик конкурентів в сфері VoIP та contact center рішень.

Проаналізуй нові статті з блогу компанії ${company}:

${blogPostsList}

Створи короткий саммарі (до 300 слів):

1. **Основні теми:** Які напрямки покриває компанія в своїх публікаціях
2. **Нові функції:** Чи згадуються нові продукти або фічі
3. **Маркетингова стратегія:** На що робиться акцент (ціна, якість, інновації)
4. **Цільова аудиторія:** Для кого призначений контент
5. **Insights:** Що може бути корисним для нашої компанії

Формат відповіді: structured markdown з чіткими секціями.
`;

return {
  prompt: prompt,
  company: company,
  timestamp: new Date().toISOString()
};
```

## Tips для кращих результатів

1. **Будьте специфічними:** Чим детальніший промпт, тим кращі результати
2. **Додавайте контекст:** Вказуйте сферу діяльності, цільову аудиторію
3. **Структуруйте відповідь:** Просіть конкретний формат (markdown, JSON, bullet points)
4. **Вказуйте довжину:** Обмежуйте кількість слів або символів
5. **Iterative refinement:** Тестуйте та покращуйте промпти на основі результатів
6. **Температура:** 0.3-0.5 для фактичного аналізу, 0.7-0.9 для креативних insights
7. **Few-shot examples:** Додавайте приклади бажаної відповіді
