# Competitor Monitoring System

A project for regularly collecting and analyzing changes on competitor websites in the SaaS segment.

## What I built
- Built a pipeline for collecting content from blogs, pricing pages, and news sections.
- Added processing of external sources and structured reporting.
- Set up Google Sheets synchronization for result delivery.

## Stack
- Python, Anthropic API, Google Sheets API.
- BeautifulSoup4, Requests.
- ThreadPoolExecutor.

## Result
- Automated competitor monitoring instead of manual website checks.
- Data is collected in a format that is easy to analyze.
- Suitable for regular market tracking.
