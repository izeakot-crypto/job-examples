# Система моніторингу конкурентів

Проєкт для регулярного збору та аналізу змін на сайтах конкурентів у SaaS-сегменті.

## Що зробив
- Зібрав pipeline для збору контенту з блогів, цінових сторінок і новинних розділів.
- Додав обробку зовнішніх джерел і підготовку структурованих звітів.
- Налаштував синхронізацію з Google Sheets для передачі результатів.

## Стек
- Python, Anthropic API, Google Sheets API.
- BeautifulSoup4, Requests.
- ThreadPoolExecutor.

## Результат
- Автоматизовано моніторинг конкурентів замість ручної перевірки сайтів.
- Дані збираються у форматі, зручному для аналітики.
- Сценарій придатний для регулярного ринкового трекінгу.
