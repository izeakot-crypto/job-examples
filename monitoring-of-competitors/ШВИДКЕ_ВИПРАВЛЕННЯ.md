## ✅ КРОК 2: Fetch Website1 (1 хвилина)

1. Знайти node **Fetch Website1**
2. Scroll вниз до розділу **Headers**
3. Натиснути **Add Header**
4. Вставити:

```
Name: User-Agent
Value: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36
```

5. **Add Header** ще раз:

```
Name: Accept
Value: text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,*/*;q=0.8
```

6. **Add Header** третій раз:

```
Name: Accept-Language
Value: en-US,en;q=0.9
```

7. Scroll до **Options** → **Redirect** → **Follow Redirects**: `true`
8. **Save**

**Що дає:**
- ✅ Сайти більше не блокують requests
- ✅ HTML з'являється в body
- ✅ Redirects обробляються (www → non-www, http → https)

## ✅ КРОК 5: AI Agent User Prompt (1 хвилина)

**ВІРОГІДНО ВЖЕ ВИПРАВЛЕНО**, але перевір:

1. Знайти node **AI Agent**
2. Відкрити вкладку **Chat**
3. Подивитися на поле **Text** (User Prompt)
4. **Перевірити чи починається з `=`** (якщо ні - додати)
5. **Переконатись що ВСІ вирази використовують `{{ }}` а НЕ `${ }`**

Приклад ПРАВИЛЬНО:
```
=КОНКУРЕНТНА РОЗВІДКА

Дата аналізу: {{ $now.format('YYYY-MM-DD') }}
Компанія: {{ $input.all()[0].json.company }}
```

Приклад НЕПРАВИЛЬНО:
```
КОНКУРЕНТНА РОЗВІДКА

Дата аналізу: ${new Date()}
Компанія: ${$input.all()[0].json.company}
```

Якщо неправильно - скопіювати вміст з файлу `AI_AGENT_USER_PROMPT_FIXED.txt`
