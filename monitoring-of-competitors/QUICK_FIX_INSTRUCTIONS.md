## 1️⃣ ВИПРАВЛЕННЯ "Append New Row1" (КРИТИЧНО!)

**Де:** Знайдіть ноду **"Append New Row1"** → клікніть → відкрийте налаштування

**Що робити:**
1. Прокрутіть до **"Columns to Send"** → **"Define Below"**
2. Для КОЖНОГО поля замініть expression:

### ДО виправлення:
```
={{ $json['Дата'] }}
```

### ПІСЛЯ виправлення (ДОДАЙТЕ `|| fallback` ДО КОЖНОГО ПОЛЯ):

Скопіюйте ці expressions і замініть у відповідних полях:

```javascript
// Поле "Дата":
={{ $json['Дата'] || new Date().toISOString().split('T')[0] }}

// Поле "Компанія":
={{ $json['Компанія'] || 'Unknown' }}

// Поле "URL":
={{ $json['URL'] || '-' }}

// Поле "Нові фічі":
={{ $json['Нові фічі'] || '-' }}

// Поле "Проблеми":
={{ $json['Проблеми'] || '-' }}

// Поле "Інсайти з коментарів":
={{ $json['Інсайти з коментарів'] || '-' }}

// Поле "Новини (з останньої перевірки)":
={{ $json['Новини (з останньої перевірки)'] || '-' }}

// Поле "Статті в блозі (з останньої перевірки)":
={{ $json['Статті в блозі (з останньої перевірки)'] || '-' }}

// Поле "YouTube активність":
={{ $json['YouTube активність'] || '-' }}

// Поле "Facebook активність":
={{ $json['Facebook активність'] || '-' }}

// Поле "LinkedIn активність":
={{ $json['LinkedIn активність'] || '-' }}

// Поле "Згадки на агрегаторах":
={{ $json['Згадки на агрегаторах'] || '-' }}

// Поле "Кількість згадок в соцмережах":
={{ $json['Кількість згадок в соцмережах'] || '0' }}

// Поле "Болі клієнтів з коментарів":
={{ $json['Болі клієнтів з коментарів'] || '-' }}

// Поле "Хотілки клієнтів з коментарів":
={{ $json['Хотілки клієнтів з коментарів'] || '-' }}

// Поле "AI Summary":
={{ $json['AI Summary'] || '-' }}

// Поле "isNewEntry":
={{ $json.isNewEntry || 'true' }}
```

**ВАЖЛИВО:** Замініть expression в КОЖНОМУ полі!

## 3️⃣ ВИПРАВЛЕННЯ "AI Agent1" (Прискорення)

**Що робити:** Замініть ВЕСЬ промпт на цей (скопіюйте повністю):

```
Ти - Senior Business Analyst в сфері VoIP/Contact Center. Проаналізуй компанію і поверни ТІЛЬКИ valid JSON.

ДАНІ:
{{ JSON.stringify($input.all(), null, 2) }}

ЗАВДАННЯ:
Створи детальний аналіз компанії. Якщо даних мало - зроби обґрунтовані висновки на основі наявної інформації.

ФОРМАТ ВІДПОВІДІ (БЕЗ ```json markdown):
{
  "company": "назва компанії",
  "url": "URL",
  "youtubeActivity": "опис активності або 'Немає YouTube каналу'",
  "linkedinActivity": "LinkedIn профіль або 'Не знайдено'",

  "reviewInsights": "аналіз відгуків клієнтів (якщо є)",
  "news": ["новини компанії або 'Новини не знайдено'"],
  "blogArticles": [{"title": "назва статті", "date": "YYYY-MM-DD", "summary": "короткий опис"}],

КРИТИЧНО ВАЖЛИВО:

3. Якщо даних немає - напиши явно (наприклад "Дані відсутні")

## 4️⃣ ДОДАТКОВО: Покращення "Format for Sheets3"

**Де:** Знайдіть ноду **"Format for Sheets3"** → відкрийте Code Editor

**Що робити:** В самому ПОЧАТКУ коду (після `const ai = data.aiAnalysis || {};`) додайте логування:

```javascript
// ДОДАЙТЕ ЦІ РЯДКИ ПІСЛЯ const ai = data.aiAnalysis || {};
console.log('=== FORMAT DEBUG ===');
console.log('Company:', data.company);
console.log('Features count:', (ai.newFeatures || []).length);
console.log('Problems count:', (ai.problems || []).length);
console.log('AI Summary length:', (ai.summary || '').length);
console.log('====================');
```

Це допоможе діагностувати проблеми в execution log.
