# Gemini JSON Translator

Інструмент для автоматизованого перекладу великих масивів JSON-даних (наприклад, локалізація застосунків) за допомогою Google Gemini API.

## Функціонал
- **Smart Translation**: Переклад значень JSON зі збереженням структури ключів.
- **Resume Capability**: Скрипт дозволяє продовжувати переклад з місця зупинки, що критично для великих файлів.
- **Sync Tool**: Модуль для синхронізації перекладених файлів з оригіналом (додавання нових ключів без перекладу заново всього файлу).
- **Monitoring**: Bash-скрипт для автоматичного моніторингу та перезапуску процесу перекладу.

## Стек
- Python, Google Generative AI SDK (Gemini).
- Shell scripting (для автоматизації).
