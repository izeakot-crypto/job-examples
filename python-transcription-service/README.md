# Сервіс транскрибації

Мікросервіс для перетворення аудіо в текст, орієнтований на обробку записів телефонних розмов.

## Що зробив
- Побудував сервіс для транскрибації аудіо у текст.
- Додав механізм відправлення результатів у зовнішні системи.
- Підготував основу для чергової обробки великої кількості файлів.

## Стек
- Python.
- ASR/LLM-інтеграції.
- Webhook-інтеграції.

## Результат
- Отримано сервіс, який можна підключати до телефонних або внутрішніх процесів.
- Автоматизовано передачу результатів без ручного копіювання.
- Сервіс можна масштабувати під більші обсяги обробки.
