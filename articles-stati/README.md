# Система автоматизації контенту

Проєкт показує повний цикл створення та публікації контенту: від генерації тем до виходу статті на WordPress.

## Що зробив
- Побудував пайплайн генерації, перевірки та доопрацювання контенту.
- Додав інтеграції для збереження прогресу, даних і результатів у базі.
- Налаштував автоматичну публікацію готових матеріалів у WordPress.

## Стек
- Python, FastAPI, Uvicorn.
- Anthropic API, Google Gemini API.
- Supabase, Pydantic, asyncio.

## Результат
- Отримано керований контент-процес без ручного складання кожного етапу.
- Зменшено кількість проміжних ручних дій під час підготовки статей.
- Реалізовано структуру, яку можна масштабувати на багатомовні сценарії.
